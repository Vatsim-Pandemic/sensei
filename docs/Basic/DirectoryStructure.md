## Directory Structure.

This is the Default Directory Structure:

```
├── commands                   
│   ├── Group 1
|   |   └── // Command Files Belonging to Group 1 
│   ├── Group 2
|   |   └── // Command Files Belonging to Group 2 
│   └── _drafts
|       └── // Draft Command Files that are ignored by the bot.
├── index.js
└── package.json
```
